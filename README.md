# 🎨 User Interface Theme with Secure Login System

## 📌 Overview
This project is a Python-based GUI application developed to improve the visual design and usability of software interfaces. It focuses on customizable themes, user interaction, and security features such as multi-factor authentication and camera-based privacy controls.

The system allows users to switch between themes, interact with a clean interface, and securely access the application using modern authentication methods. :contentReference[oaicite:2]{index=2}

---

## 🎯 Objectives
- Improve UI design using customizable themes (colors, layout, styles)
- Enhance user experience with a clean and interactive interface
- Implement secure login using multi-factor authentication
- Provide camera access with privacy features like face blur
- Ensure smooth performance and responsiveness :contentReference[oaicite:3]{index=3}

## ⚙️ Features

### 🎨 UI & Theme Customization
- Dark Mode (reduces eye strain)
- Light Mode (clean and readable interface)
- Dynamic theme switching
- Custom color settings

### 🔐 Security Features
- Multi-Factor Authentication (OTP + QR Code)
- Secure login system
- Gmail OTP integration

### 📷 Camera Module
- Live camera access
- Face blur feature for privacy
- Start/Stop camera controls

### 🖥️ User Dashboard
- Interactive dashboard after login
- Easy navigation and controls
- Integrated system modules

---

## 🏗️ System Architecture

The system is divided into modular components:

- **GUI Layer** → Handles user interface and display
- **Theme Management Module** → Controls themes, colors, styles
- **User Interaction Module** → Manages inputs and navigation
- **Authentication Module** → Implements MFA security
- **Camera Module** → Handles camera and blur functionality :contentReference[oaicite:4]{index=4}
