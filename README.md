# 🎨 User Interface Theme with Secure Login System

## 📌 Overview

This project is a Python-based GUI application developed to improve the visual design and usability of software interfaces. It focuses on customizable themes, user-friendly interaction, and built-in security features such as multi-factor authentication and camera-based privacy controls. 

The system allows users to switch between different UI themes, securely log in, and interact with an intuitive dashboard designed for smooth performance.

---

## 🎯 Objectives

* Improve UI appearance using customizable colors, layouts, and styles
* Enhance user experience through clean and interactive design
* Implement secure login using multi-factor authentication
* Provide camera access with privacy features like face blur
* Ensure responsiveness and smooth system performance 

---

## ⚙️ Features

### 🎨 UI & Theme Customization

* Dark Mode (reduces eye strain)
* Light Mode (clean and readable interface)
* Dynamic theme switching
* Custom color configuration

### 🔐 Security Features

* Multi-Factor Authentication (OTP + QR Code)
* Secure login system
* Gmail OTP integration

### 📷 Camera Module

* Live camera access
* Face blur feature for privacy
* Start/Stop camera controls

### 🖥️ User Dashboard

* Interactive dashboard after login
* Easy navigation and user controls
* Integrated system modules

---

## 🏗️ System Architecture

The project follows a modular design:

* **GUI Layer** → Handles user interface and display
* **Theme Management Module** → Controls themes, colors, and styles
* **User Interaction Module** → Manages user input and navigation
* **Authentication Module** → Implements multi-factor security
* **Camera Module** → Handles camera access and blur functionality

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/Ali4156/ui-theme-project.git
```

2. Open the project folder:

```
cd ui-theme-project
```

3. Install dependencies (if required):

```
pip install -r requirements.txt
```

4. Run the application:

```
python main.py
```

---

## 🔑 System Workflow

1. User launches the application
2. Login screen appears
3. User enters credentials
4. Multi-factor authentication (OTP or QR) is required
5. After verification, dashboard loads
6. User can switch themes, access camera, and control privacy settings
